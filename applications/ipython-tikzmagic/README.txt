# Installation

```sh
pip install git+git://github.com/mkrphys/ipython-tikzmagic.git
```

The package requires a working LaTeX installation, ImageMagick and pdf2svg.

One needs to update the default $PATH used in tikzmagic.py. I modified this file, which you can replace in 

/usr/local/Caskroom/miniconda/base/envs/datascience-py3p8-r3p6p0-env/lib/python3.8/site-packages/tikzmagic.py
